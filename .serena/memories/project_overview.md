SLDB is a Python library and CLI for treating Markdown as structured, typed, reversible data using Pydantic models (`StructuredNLDoc`). It supports extracting typed payloads from Markdown, rendering Markdown from typed data, validating roundtrip/idempotency behavior, and a newer `.sldb/` store system for tracking model contracts and document instances via YAML indexes and a Merkle-style hash cascade. Code lives mainly under `src/sldb/`; tests live under `tests/`.
