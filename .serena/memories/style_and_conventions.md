Python project with strong type-hint usage, small helper functions, and Pydantic models for data contracts. Naming is mostly snake_case for functions/modules and PascalCase for models/classes. Public APIs are re-exported from `src/sldb/__init__.py`. `StructuredNLDoc` subclasses are expected to define `__template__` and every Pydantic field must have a non-empty `description`. Code style is concise and functional; docstrings are sparse, so follow nearby code rather than introducing heavy documentation unless needed.
