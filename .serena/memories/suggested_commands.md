Install package/dev deps: `pip install -e .[dev]`
Run tests: `pytest`
Run a specific test file: `pytest tests/test_standalone.py`
Run store tests: `pytest tests/store`
Run CLI entrypoint: `python -m sldb --help`
Installed CLI: `sldb --help`
Build package: `python -m build`
Git status: `git status --short --branch`
Recent history: `git log --oneline --decorate -5`
